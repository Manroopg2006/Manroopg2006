## 🐋 Flagship Project — OrcaPath AI

> Real-time marine bioacoustics system for detecting and analyzing killer whale vocalizations from hydrophone data.

### Key Capabilities
- 🎧 Mel spectrogram ML pipeline for underwater audio analysis  
- 🧠 Random Forest classifier for orca vs non-orca detection  
- 📍 Spatiotemporal mapping across Pacific Northwest hydrophones  
- 📊 Behavioral clustering + temporal sequence analysis  
- 📡 Supports historical + multi-station acoustic datasets

- Real-world environmental sensor data (hydrophones)
- End-to-end ML pipeline (raw audio → features → inference → visualization)
- Behavioral + statistical analysis layered on top of ML
- Bridges **research + production system design**
